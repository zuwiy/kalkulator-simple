# Kalkulator - Web Calculator

A simple and elegant web-based calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clear all (AC) function
- Delete last digit (DEL) function
- Support for decimal numbers (using comma as decimal separator)
- Responsive design centered on the page
- Clean and modern UI with rounded buttons

## Project Structure

```
/workspace
├── index.html    # Main HTML structure
├── style.css     # Styling and layout
└── script.js     # Calculator logic and functionality
```

## Usage

1. Open `index.html` in any modern web browser
2. Click the buttons to perform calculations
3. Use **AC** to clear the entire display
4. Use **DEL** to delete the last character
5. Press **=** to calculate the result

## Button Functions

| Button | Description |
|--------|-------------|
| AC     | Clear all input |
| DEL    | Delete last character |
| ÷      | Division |
| ×      | Multiplication |
| -      | Subtraction |
| +      | Addition |
| =      | Calculate result |
| 0-9    | Number input |
| ,      | Decimal separator |

## Technologies Used

- **HTML5** - Structure of the calculator
- **CSS3** - Styling with flexbox layout
- **JavaScript (ES6)** - Interactive functionality
